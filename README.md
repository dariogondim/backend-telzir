
## Description
  Api telzir. comparative costs with or without a plan of minutes

## Installation

```bash
$ yarn
```

## Running the app

```bash
# start
$ nest start
$ yarn start
$ npm start

#Projects runs/test with specific v12.18.4 (npm v6.14.6), but it should work in other versions

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
